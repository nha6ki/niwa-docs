JupyterLab¶

Docker¶

JupyterLabをDockerで使用する方法．

docker run --rm -p 8888:8888 -e JUPYTER_ENABLE_LAB=yes -v `pwd`:/home/jovyan/work --name jupyter jupyter/datascience-notebook
